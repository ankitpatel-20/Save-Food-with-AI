# PureHarvest AI - Save Food with AI

PureHarvest AI is a smart food resource allocation and waste reduction platform. It leverages AI to optimize the logistics of food donations, connecting surplus food donors (restaurants, grocery stores) with recipients in need (NGOs, shelters).

## 🌍 Overview
The platform addresses the critical challenge of food waste by streamlining the "Last Mile" of food recovery. Using the Gemini Logistics Engine, the app prioritizes deliveries based on food perishability and community urgency.

## ✨ Key Features
- **Smart Allocation Engine**: Automatically matches donor supplies with recipient needs using AI-driven priority scoring.
- **Real-time Logistics Monitoring**: A visual command center for tracking batches, donor sources, and delivery statuses.
- **Interactive Mapping**: Integrated GIS visualization for donor hubs and recipient locations.
- **Impact Forecasting**: Live analytics for waste diversion rates and potential beneficiary impact.
- **PureHarvest AI Chat Agent**: An intelligent assistant for food safety guidelines, donation best practices, and system support.
- **Emergency Protocols**: Automated detection of high-spoilage risks with one-click express courier dispatch.

## 🛠 Tech Stack
- **Frontend**: React 18 + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **AI Engine**: Google Gemini (via `@google/genai`)

## 🚀 Getting Started
1. Clone the repository.
2. Create a `.env` file and add your `GEMINI_API_KEY`.
3. Install dependencies: `npm install`
4. Run the development server: `npm run dev`

## 📊 Impact
By optimizing the chain from donation to delivery, PureHarvest AI helps reduce CO2 emissions from food rotting in landfills and ensures fresh food reaches those who need it most, faster.
